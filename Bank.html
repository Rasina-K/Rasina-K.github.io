Web Scraping Using Beautiful Soup

Project Synopsis

In this project, we aim to employ web scraping techniques, utilizing the Beautiful Soup library, to gather a comprehensive list of the largest banks in the United States. The project focuses on extracting relevant information from web pages, specifically targeting data related to the size and ranking of banks based on their assets.

Features

The scope of the project is limited to web scraping for publicly available information on the largest U.S. banks. The focus is on automating the extraction process to create a dataset that can be utilized for various analytical purposes.
